Surat 'Cinta' untuk Mentorku

Kehadapan mentorku, moga tuan sentiasa sihat sahaja. Untuk makluman tuan, saya ada beberapa perkara yang ingin saya sampaikan:

  • Pertamanya saya ingin mengucapkan ribuan terima kasih kerana sudi menjadi mentor saya
  • Saya bukanlah insan sempurna, melainkan untuk menjadi seorang programmer yang terhebat macam tuan
  • Saya berikrar untuk menjadi pelajar yang rajin dan cuba untuk tidak ponteng.
  • Jika saya ada buat silap dan salah, mohon tuan tidak memarahi saya tetapi tegurlah saya secara berhemah
  • Saya doakan tuan berjaya di dunia dan di akhirat
  • Saya doakan juga moga ikatan ukhuwwah kita semua sampai ke syurga insyaAllah

Kirim salam saya kepada kucing tuan (jika ada).
